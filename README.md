# Retro Arcade Hub

A responsive retro arcade dashboard built with React, Tailwind CSS, and Framer Motion. Features fully playable canvas games, local high-score tracking, achievements, and an in-app lofi radio stream.

👉 **[Play the Live Demo here!](https://retroarcade-aanvi.vercel.app/)**

## Playable Games
- **Neon Snake**: Classic snake with keyboard controls and speed progression.
- **Retro Pong**: Standard Pong against a simple computer opponent.
- **Space Invaders**: Defend the grid from waves of invaders. Move with Arrow Keys and shoot with Space.
- **Flappy Clone**: Dodge obstacles by controlling gravity. Press Space to jump.
- **Tetris**: Rotate and drop falling tetrominoes to clear rows.

## Features
- **Global Lofi Radio**: A toggleable, skippable audio player integrated into the navigation bar streaming Chillhop and Lounge channels.
- **Dynamic Leaderboard**: Local high scores automatically save and rank you on the local "Hall of Fame".
- **SPA Routing Support**: Configured with redirection headers for smooth hosting on Vercel and Netlify.

## Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/aanvi076/retro-arcade.git
   cd retro-arcade
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
